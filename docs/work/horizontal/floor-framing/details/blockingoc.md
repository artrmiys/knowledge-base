# Blocking oc

## Formulas

Use project-specific Excel syntax, but the logic from notes is:

```text
Flat 48" o.c.     = G * 12 / 48 * 2 * 1.1 / D, ceiling to 1
Diagonal 48" o.c. = G * 12 / 48 * 2.5 * 1.1 / D, ceiling to 1
```
