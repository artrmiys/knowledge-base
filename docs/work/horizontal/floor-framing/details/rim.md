# Rim Board

## Rules

- Do not write `1-3/4 LVL Rim` unless LVL is explicitly called out or needed by
  detail.
- Default output can be `11-7/8" Rim`.
- If LSL is specified, write `11-7/8" LSL Rim`.
- Rim factor is 1.05 in LFT.
- Roof TJI needs rim too.

## Typical Products

| Product | When |
| --- | --- |
| 1-1/8" OSB Rim | Common thin rim |
| 1-1/4" or 1-1/2" LSL Rim | When LSL or stronger rim is specified |
| 1-3/4" LVL Rim | Deck/corridor frame or explicit detail |

## Check

If product is assumed, say `assumed` in the note.
