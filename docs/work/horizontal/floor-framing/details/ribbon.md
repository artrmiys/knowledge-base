# Ribbon Board

## Count

- Ribbon board where joists/trusses require it and details show it.

## Rule

Top chord bearing trusses do not need 2x4 ribbon board. In those conditions,
use blocking between trusses instead, often `(2) 2x6` over 2x6 bearing walls.

## Check

- Do not confuse ribbon board, rim, and blocking.
- If trusses hang over walls, check whether blocking or drywall ledger is the
  actual material.
