# Duplicate of Gable

This page is kept because the current takeoff tool tree has this item. Use it only
when the job template has a separate duplicated gable bucket.

## Rule

Do not double-count gable sheathing. If this item is used, add a note explaining
which gable area is here and which area is in regular Gable Sheathing.
