# Duplicate of Gable

Эта страница оставлена, потому что current takeoff tool tree имеет этот item. Используй её только
когда job template имеет отдельный duplicated gable bucket.

## Правило

Не double-count gable sheathing. Если этот item используется, добавь note, объясняющую
which gable area is here and which area is in regular Gable Sheathing.
