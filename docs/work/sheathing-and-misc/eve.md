# Eve

## Count

- Eve framing, soffit frames, soffit plywood/gypsum, subfascia, and trim where
  shown.

## Rules

- Eve soffit frames can be 2x6, not 2x4, if drawings show it.
- If 30% or more of soffit is beadboard, count the full soffit SQFT as beadboard
  when using that local rule.

## Check

- RCP and architectural sections can show soffits missing from structural plans.
