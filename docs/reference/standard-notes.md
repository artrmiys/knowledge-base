# Standard notes

Библиотека канонических takeoff-notes из воркбука (лист `Standarts`, ~700
формулировок). Это **переиспользуемые заметки-допущения**: когда деталь не
указана, ставишь стандартную note вместо догадки.

!!! tip "Принцип"
    Note всегда содержит: что **assumed** + откуда (`per page S101`) +
    `verify`. Видимое допущение лучше, чем тихая догадка. Конкретную ссылку
    на лист подставляй из своего проекта.

- `Note: 5/8" and 3/4" CDX Roof Sheathing per Arch; 1/2" CDX per Structural; Assumed per Arch`
- `Note: 7/16" Zip indicated on one page only; Assumed Wall Sheathing 1/2" Ply; verify`
- `Note: 15/32" OSB Wall Sheathing contradicts R-6 Zip per A4.1; verify`
- `Note: 1 1/2" R6.0 Wall Sheathing; Assumed 7/16" OSB; verify`

!!! warning "Arch vs Structural"
    Sheathing на Arch и Structural часто противоречат. Не выбирай молча —
    `Assumed per Structural; verify` (или per Arch, по правилу проекта).
