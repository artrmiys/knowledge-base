# Takeoff item labels

Канонический словарь Label'ов из воркбука (лист `Standarts`, колонка A). Это
**имена строк takeoff** — пиши их ровно так, как здесь, чтобы output, formulas
и review совпадали.

!!! tip "Зачем фиксировать имена"
    `Beam (2) Dropped`, `Plates Ext (2) dbl top`, `Joist 16" o.c.` — это
    канонические Label'ы. Свободные формулировки ломают sort/sum-макросы
    ([Trim macros](../work/exterior-trims/macros.md)) и сверку.
